# 🥋 DevOps_Dojo

> *"Master the art of DevOps, one commit at a time"*

Welcome to **DevOps_Dojo** — Your personal training ground for cloud infrastructure mastery! This repository is a collection of hands-on DevOps practices, Terraform configurations, and interview preparation materials designed to sharpen your skills in modern cloud engineering.

---

## 🎯 What's Inside?

### 📦 **Terraform_Practice** 
Your lab for Infrastructure-as-Code excellence. Here you'll find:
- **Storage**: Azure Resource Group automation with `for_each` magic ✨
- **Storage_Account_Use**: Complete storage account deployments
- More infrastructure patterns coming soon!

### 🎤 **Interview**
Ace your DevOps interviews with curated questions, scenarios, and solutions.

---

## 🚀 Quick Start

### Prerequisites
```bash
- Terraform (v1.0+)
- Azure CLI
- Git
```

### Setup & Deploy
```bash
# Navigate to any practice directory
cd Terraform_Practice/Storage

# Initialize Terraform
terraform init

# Preview your changes
terraform plan

# Deploy to Azure
terraform apply
```
